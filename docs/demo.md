# Demo Script

## Goal

Show that healthcare-style data can be prepared client-side, evaluated privately, and anchored to an immutable ledger without exposing plaintext.

## Demo Flow

1. Open the React app and connect a wallet.
2. Explain the privacy badges and note that the client keeps the secret key locally.
3. Upload or prepare a healthcare record locally.
4. Show encrypted payload submission to the HE service.
5. Publish or reference the contribution on the ledger.
6. Request a prediction and show the returned encrypted result.
7. Show the ledger event timeline with model version and contribution hash.
